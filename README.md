# Memory

## Requirements

Install `yarn`, read how [here](https://yarnpkg.com/en/docs/install).

## Installation

Run the following commands:

1. `git clone git@github.com:akinorew/memory.git`
2. `cd memory`
3. `yarn install`

To start the application you can now run: `yarn start`
